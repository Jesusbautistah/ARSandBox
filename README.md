# ARSandBox
Augmented reality sand box designed to work with the IDAV -UC Davis- [software](http://idav.ucdavis.edu/~okreylos/ResDev/SARndbox/).

Included STEP DXF and BOM sources.

Spanish manual by Angel Valverde
